html , css چیست؟

html چیست؟

Html یا hypertext markup language زبان طراحی و توسعه وب است. اما یک «زبان برنامه نویسی» نیست! html یک زبان تگ گذاری ست. ما با HTML بخش های مختلف صفحات وب و به قولی اسکلت صفحات وبمون رو درست می کنیم. زبان تگ گذاری زبانی ست که دستورات و کد های آن بین دو علامت < و > قرار می گیرد. کد های «زبان تگ گذاری» از جنس متن یا Text هست و زبان برنامه نویسی نیستند. اما برای توسعه وب لازم هستند. زیرا فقط با html و css می توان صفحات وب را «طراحی» کرد.

Css چیست؟

CSS مخفف cascading style sheets به معنی و یک «زبان style sheet» ست. اما کاربرد آن متفاوت است. این زبان برای زیباسازی صفحه وب و استایل دهی به آن استفاده می شود. به این شکل که ما با CSS به کمک انتخابگر ها یا selector ها یک المان یا گروهی از المان ها رو انتخاب می کنیم و بر روی آنها استایل هایی رو اعمال می کنیم. همیشه از CSS و HTML در کنار هم استفاده می شود.
